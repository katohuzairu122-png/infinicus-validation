# INFINICUS ENGINE v3 — Business Reality Simulator

Simulate your business idea across 30 virtual days with a statistical engine, Monte Carlo risk analysis, and a **Go / Modify / Stop** verdict. No signup. No API key required.

## Features

- Monte Carlo risk simulation over a 30-day business timeline
- Go / Modify / Stop decision verdict based on your inputs
- PWA-installable — works offline after first load
- Zero backend — runs entirely in the browser

## Getting Started

Open `index.html` directly in a browser, or visit the deployed version via GitHub Pages.

## Stack

- Vanilla HTML / CSS / JavaScript
- Supabase (optional analytics via `supabase.js`)
- GitHub Actions for deployment (`deploy.yml`)
