# Architecture

## Upstream

AI Decision Intelligence publishes a governed decision package.

## Approved Business Action

ABA-01 through ABA-24 convert the decision into:

- validated action
- approval
- approved contract
- bounded execution plan
- assigned tasks
- reserved resources
- controlled schedule
- connector invocation
- dry run
- controlled execution
- rollback and compensation handling
- execution evidence
- completion verification
- monitoring contract
- Outcome Monitoring and Continuous Learning handoffs

## ABA-25 role

ABA-25 does not replace any block. It provides:

- assembly
- dependency validation
- orchestration
- readiness diagnostics
- deployment evidence
- terminal-handoff validation
