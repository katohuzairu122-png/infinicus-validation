# INFINICUS ABA-14 — Responsible Actor and Task Assignment Engine

Version: 1.0.0

ABA-14 assigns governed execution tasks to eligible people, teams, departments, systems, or external providers.

Public API:

`window.INFINICUS.ABA.responsibleActorTaskAssignmentEngine`

## Capabilities

- Actor registry
- Team registry
- Capability registry
- Availability records
- Task eligibility checks
- Workload checks
- Separation-of-duties checks
- Primary and backup assignment
- Acceptance and rejection workflow
- Reassignment
- ABA-15 resource-reservation handoff
- IndexedDB persistence
