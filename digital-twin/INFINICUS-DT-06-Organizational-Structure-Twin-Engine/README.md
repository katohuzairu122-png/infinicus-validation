# INFINICUS DT-06 — Organizational Structure Twin Engine

Version: 1.0.0

DT-06 interprets the generic DT-05 entity graph as an organizational structure containing units, teams, roles, reporting lines, responsibilities, decision rights, and accountability paths.

## Capabilities

- Organizational unit registry
- Role registry
- Team registry
- Position and assignment registry
- Reporting-line validation
- Responsibility and accountability mapping
- Decision-right mapping
- Span-of-control analysis
- Management-layer analysis
- Vacant-role and overloaded-manager detection
- Organizational chart generation
- DT-07 financial-state handoff
- IndexedDB persistence
- Demo interface and automated tests

## Dependencies

- DT-01 through DT-05

## Public API

`window.INFINICUS.DT.organizationalStructureEngine`
