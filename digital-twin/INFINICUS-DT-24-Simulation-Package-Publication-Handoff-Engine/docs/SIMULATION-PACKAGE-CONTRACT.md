# Simulation Package Contract

## Manifest

- package ID
- package version
- contract version
- business ID
- twin ID
- scenario ID
- historical snapshot reference
- historical checksum
- scenario checksum
- horizon
- time step
- seed
- state and evidence counts
- validation result

## State Partitions

- actual state
- assumed state
- simulated state

These partitions must never be merged without labels.
