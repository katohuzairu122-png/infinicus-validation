# INFINICUS DT-01 — Digital Twin Core Runtime and Registry

Version: 1.0.0

DT-01 provides the foundational runtime for the Business Digital Twin layer.

## Capabilities

- Digital Twin namespace and runtime initialization
- Standard success and failure envelopes
- Service, route, entity, state, event, schema, and adapter registries
- Event bus
- Twin lifecycle status management
- Correlation and causation identifiers
- Immutable cloning helpers
- Runtime diagnostics
- 24-block Digital Twin manifest
- Browser-global dependency-free JavaScript
- Demo interface and automated tests

## Public API

`window.INFINICUS.DT.runtime`
