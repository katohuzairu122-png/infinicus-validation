## Block namespace

`window.INFINICUS.ADI.blocks["ADI-22"]`

## Factory

`createRedTeamEngine(options)` — returns the ADI-22 engine/service object (`blockId: "ADI-22"`, `version: "1.0.0"`).

## attachToADIRuntime

`attachToADIRuntime(runtime, options)` — validates dependencies, registers the service and routes, returns a result envelope.
