# ADI-14 API

## Block namespace

`window.INFINICUS.ADI.blocks["ADI-14"]`

## Factory

`createFeasibilityFilter(options)` — returns the ADI-14 engine/service object (`blockId: "ADI-14"`, `version: "1.0.0"`).

## attachToADIRuntime

`attachToADIRuntime(runtime, options)` — validates dependencies, registers the service and routes, returns a result envelope.
