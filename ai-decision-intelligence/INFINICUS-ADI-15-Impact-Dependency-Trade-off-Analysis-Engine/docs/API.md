# ADI-15 API

## Block namespace

`window.INFINICUS.ADI.blocks["ADI-15"]`

## Factory

`createImpactAnalysisEngine(options)` — returns the ADI-15 engine/service object (`blockId: "ADI-15"`, `version: "1.0.0"`).

## attachToADIRuntime

`attachToADIRuntime(runtime, options)` — validates dependencies, registers the service and routes, returns a result envelope.
