# ADI-17 API

## Block namespace

`window.INFINICUS.ADI.blocks["ADI-17"]`

## Factory

`createRiskAssessmentEngine(options)` — returns the ADI-17 engine/service object (`blockId: "ADI-17"`, `version: "1.0.0"`).

## attachToADIRuntime

`attachToADIRuntime(runtime, options)` — validates dependencies, registers the service and routes, returns a result envelope.
