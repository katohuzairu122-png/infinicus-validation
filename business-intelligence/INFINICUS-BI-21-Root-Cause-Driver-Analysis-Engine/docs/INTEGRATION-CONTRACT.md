# BI-20 to BI-21 Contract

BI-21 receives:

- root-cause handoff ID,
- detection run ID,
- prioritized signals,
- confidence scores,
- evidence,
- investigation queue,
- domains,
- trends,
- variances,
- benchmarks,
- correlation ID.

# BI-21 to BI-22 Contract

BI-21 provides:

- root-cause analysis ID,
- investigation case,
- ranked drivers,
- evidence confidence,
- cause graph,
- unresolved hypotheses,
- supporting context,
- correlation ID.
