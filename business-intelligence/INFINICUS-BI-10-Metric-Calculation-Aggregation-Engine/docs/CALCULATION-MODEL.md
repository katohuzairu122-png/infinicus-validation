# Calculation Model

## Base Metrics

Calculated directly from warehouse records.

## Derived Metrics

Calculated from dependent metric results.

## Ratio Metrics

`numerator / denominator`

## Rate Metrics

`numerator / denominator × 100`

## Target Metrics

Return the governed target value for comparison.

## Grouping

Metrics may be grouped by:

- business dimensions,
- day,
- month,
- year,
- all time.
