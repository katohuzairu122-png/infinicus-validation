# BI-19 to BI-20 Contract

BI-20 receives:

- anomaly handoff ID,
- analysis run ID,
- consolidated domains,
- trends,
- variances,
- benchmarks,
- domain ranking,
- correlation ID.

# BI-20 to BI-21 Contract

BI-20 provides:

- detection run ID,
- prioritized signals,
- detection confidence,
- evidence,
- investigation queue,
- supporting domains, trends, variances, and benchmarks,
- correlation ID.
