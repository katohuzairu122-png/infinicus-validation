# BI-07 to BI-08 Contract

BI-08 receives:

- warehouse handoff ID,
- transformed records,
- transformation lineage,
- failed transformation records,
- dataset contract ID,
- correlation ID.

# BI-08 to BI-09 Contract

BI-08 provides:

- warehouse dataset ID,
- warehouse load ID,
- snapshot ID,
- dataset type,
- grain,
- primary key fields,
- partition fields,
- row count,
- dataset contract ID.
