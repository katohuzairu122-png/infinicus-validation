# BI-10 to BI-15 Contract

BI-15 receives:

- intelligence handoff ID,
- calculation run ID,
- warehouse snapshot ID,
- governed operations metric results,
- process and location groups,
- targets,
- threshold status,
- correlation ID.

# BI-15 to BI-19 Contract

BI-15 provides:

- operations analysis ID,
- operations profile,
- operations health score,
- process ranking,
- location ranking,
- bottleneck indicators,
- warning and opportunity signals,
- correlation ID.
