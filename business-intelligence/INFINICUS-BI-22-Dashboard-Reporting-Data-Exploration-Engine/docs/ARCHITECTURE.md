# Architecture

## Processing Sequence

1. Register dashboard and report definitions.
2. Define widgets, filters, audiences, and drill paths.
3. Load BI-21 reporting handoff.
4. Build executive and investigation summaries.
5. Render widgets from governed context.
6. Publish exploration datasets.
7. Generate immutable report snapshots.
8. Preserve source analysis and root-cause evidence.
9. Prepare report distribution metadata.
10. Publish handoff to BI-23.

## Responsibility Boundary

BI-22 creates reports and dashboard outputs.

BI-23 schedules and distributes them.
