# INFINICUS BI-11 — Financial Intelligence Engine

Version: 1.0.0

BI-11 interprets governed financial metrics and produces structured intelligence on revenue, cost, margin, cash flow, profitability, liquidity, burn, runway, and financial risk.

## Capabilities

- Financial metric classification
- Profitability analysis
- Margin analysis
- Cost structure analysis
- Cash-flow intelligence
- Liquidity and runway assessment
- Budget variance interpretation
- Financial health scoring
- Financial warning and risk signals
- Financial intelligence summaries
- BI-19 analysis handoff
- IndexedDB persistence
- Demo interface and automated tests

## Dependencies

- BI-01 through BI-10

## Public API

`window.INFINICUS.BI.financialIntelligenceEngine`
