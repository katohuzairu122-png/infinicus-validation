# BI-03 to BI-04 Contract

BI-04 receives:

- quality handoff ID,
- ingestion run ID,
- dataset contract ID,
- mapped records,
- mapping rejections,
- correlation ID,
- cursor and watermark metadata.

# BI-04 to BI-05 Contract

BI-04 provides:

- quality run ID,
- accepted records,
- warning records,
- quarantined records,
- quality issues,
- dataset quality score,
- correlation ID.
