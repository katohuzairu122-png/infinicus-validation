# BI-10 to BI-17 Contract

BI-17 receives:

- intelligence handoff ID,
- calculation run ID,
- warehouse snapshot ID,
- governed workforce metric results,
- team, role, and location groups,
- targets,
- threshold status,
- correlation ID.

# BI-17 to BI-19 Contract

BI-17 provides:

- workforce analysis ID,
- workforce profile,
- workforce health score,
- team ranking,
- role-risk ranking,
- location ranking,
- warning and opportunity signals,
- correlation ID.
