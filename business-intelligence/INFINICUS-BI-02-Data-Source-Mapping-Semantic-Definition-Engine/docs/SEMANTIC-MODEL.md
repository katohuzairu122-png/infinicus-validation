# Initial Canonical Semantic Model

Recommended canonical entities for the first INFINICUS Business Intelligence implementation:

## Dimensions

- Business
- Customer
- Product
- Service
- Employee
- Supplier
- Location
- Channel
- Date
- Time

## Facts

- Sale
- Payment
- Expense
- Inventory Movement
- Customer Interaction
- Marketing Activity
- Work Shift
- Operational Event

Each fact must define its grain clearly. Example:

> One row per completed sale transaction line.
