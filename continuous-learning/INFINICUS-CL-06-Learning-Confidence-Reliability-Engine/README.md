# INFINICUS CL-06 — Learning Confidence and Reliability Engine

Version: 1.0.0

CL-06 calculates governed confidence and reliability ratings for learning items before they can influence assumptions, rules, models, or decisions.

Public API:

`window.INFINICUS.CL.learningConfidenceReliabilityEngine`

## Capabilities

- Confidence-policy registry
- Evidence-confidence weighting
- Provenance-quality weighting
- Classification-confidence weighting
- Applicability-confidence weighting
- Reliability scoring
- Limitation and unclassified penalties
- Scope-restriction penalties
- Confidence and reliability bands
- Learning eligibility classification
- CL-07 conflict-detection handoff
- IndexedDB persistence
