# INFINICUS CL-05 — Applicability Scope and Context Engine

Version: 1.0.0

CL-05 determines where, when, and under which operating conditions a classified learning item may be safely applied.

Public API:

`window.INFINICUS.CL.applicabilityScopeContextEngine`

## Capabilities

- Applicability-policy registry
- Context-dimension registry
- Business, market, geography, scale, segment, channel, and time scope
- Declared-versus-observed context comparison
- Context similarity scoring
- Scope restriction generation
- Transferability classification
- Out-of-scope detection
- Limitation propagation
- CL-06 confidence and reliability handoff
- IndexedDB persistence
