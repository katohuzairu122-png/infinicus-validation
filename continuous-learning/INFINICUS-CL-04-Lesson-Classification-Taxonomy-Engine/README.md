# INFINICUS CL-04 — Lesson Classification and Taxonomy Engine

Version: 1.0.0

CL-04 classifies governed learning evidence into structured lesson categories and taxonomies for downstream applicability, conflict detection, and controlled knowledge updates.

Public API:

`window.INFINICUS.CL.lessonClassificationTaxonomyEngine`

## Capabilities

- Taxonomy-policy registry
- Taxonomy registry
- Category and subcategory registration
- Evidence-to-taxonomy classification
- Factual, contextual, and hypothesis separation
- Business-domain classification
- Learning-purpose classification
- Multi-label classification
- Classification confidence scoring
- Unclassified-item handling
- CL-05 applicability-scope handoff
- IndexedDB persistence
