# INFINICUS OM-12 — Alert and Escalation Engine

Version: 1.0.0

OM-12 converts material OM-11 threshold breaches into governed alerts and escalation workflows.

Public API:

`window.INFINICUS.OM.alertEscalationEngine`

## Capabilities

- Alert policy registry
- Severity routing
- Audience and owner assignment
- Alert creation
- Acknowledgement deadlines
- Escalation stages
- Retry and reminder scheduling
- Suppression and duplicate protection
- Alert lifecycle management
- Resolution evidence
- OM-13 attribution-evidence handoff
- IndexedDB persistence
