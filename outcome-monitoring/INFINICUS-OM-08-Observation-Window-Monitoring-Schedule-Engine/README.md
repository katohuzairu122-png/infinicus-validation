# INFINICUS OM-08 — Observation Window and Monitoring Schedule Engine

Version: 1.0.0

OM-08 converts governed baselines, targets, review cadences, and observation windows into executable monitoring schedules.

Public API:

`window.INFINICUS.OM.observationWindowScheduleEngine`

## Capabilities

- Monitoring schedule policy registry
- Observation window validation
- Cadence calculation
- Evaluation checkpoint generation
- Start, pause, resume, cancel, and expiry controls
- Grace periods
- Late-observation handling
- Schedule versioning
- Metric-level schedule assignments
- OM-09 normalization handoff
- IndexedDB persistence
