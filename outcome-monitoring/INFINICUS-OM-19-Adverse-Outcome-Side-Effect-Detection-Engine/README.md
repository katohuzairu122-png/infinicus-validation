# INFINICUS OM-19 — Adverse Outcome and Side-Effect Detection Engine

Version: 1.0.0

OM-19 identifies negative consequences, unintended outcomes, displaced costs, operational degradation, compliance exposure, and other side effects associated with the approved action.

Public API:

`window.INFINICUS.OM.adverseOutcomeSideEffectEngine`

## Capabilities

- Adverse-outcome policy registry
- Adverse metric definition registry
- Negative-effect detection
- Unintended outcome detection
- Displaced-cost detection
- Benefit-offset calculation
- Severity and materiality scoring
- Causation and attribution context
- Reversibility and persistence assessment
- Mitigation requirement classification
- OM-20 exception and missing-data handoff
- IndexedDB persistence
