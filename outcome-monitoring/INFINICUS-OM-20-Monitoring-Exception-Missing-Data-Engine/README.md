# INFINICUS OM-20 — Monitoring Exception and Missing-Data Engine

Version: 1.0.0

OM-20 detects and governs monitoring exceptions, missing observations, incomplete evidence, stale checkpoints, connector failures, schedule failures, and unresolved monitoring gaps.

Public API:

`window.INFINICUS.OM.monitoringExceptionMissingDataEngine`

## Capabilities

- Exception-policy registry
- Missing-data detection
- Missing-checkpoint detection
- Stale-observation detection
- Connector and source failure classification
- Evidence-gap classification
- Exception severity and materiality scoring
- Recovery and remediation recommendations
- Exception lifecycle management
- Waiver and acceptance controls
- OM-21 evidence and audit-trail handoff
- IndexedDB persistence
