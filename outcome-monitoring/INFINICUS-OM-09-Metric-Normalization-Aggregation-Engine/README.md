# INFINICUS OM-09 — Metric Normalization and Aggregation Engine

Version: 1.0.0

OM-09 standardizes validated observations into comparable metric values and applies governed aggregation rules.

Public API:

`window.INFINICUS.OM.metricNormalizationAggregationEngine`

## Capabilities

- Unit normalization
- Value-type coercion
- Currency and scale adapters
- Metric-level aggregation
- Latest, sum, average, minimum, maximum, count, and weighted-average modes
- Window-bound aggregation
- Duplicate protection
- Normalization evidence
- Aggregation evidence
- Confidence propagation
- OM-10 progress-calculation handoff
- IndexedDB persistence
