# INFINICUS OM-01 — Outcome Monitoring Core Runtime and Registry

Version: 1.0.0

OM-01 is the foundation block for the Outcome Monitoring layer.

Public namespace:

`window.INFINICUS.OM`

Public runtime API:

`window.INFINICUS.OM.runtime`

- Event bus

- Safe cloning

- Observation-source registry
- Monitoring-contract registry
- Outcome-state registry
- Layer diagnostics
- OM-01 through OM-25 manifest
- Health and readiness checks
- Integration boundary with ABA-24
