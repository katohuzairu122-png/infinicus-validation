# INFINICUS OM-17 — Outcome Confidence and Reliability Engine

Version: 1.0.0

OM-17 produces the governed confidence and reliability rating for each monitored outcome.

Public API:

`window.INFINICUS.OM.outcomeConfidenceReliabilityEngine`

## Capabilities

- Confidence-policy registry
- Evidence-quality weighting
- Attribution-confidence weighting
- Causation-confidence weighting
- Confounder penalty
- Sample sufficiency
- Source reliability
- Temporal coverage
- Confidence score
- Reliability score
- Confidence-band classification
- Reliability-band classification
- OM-18 benefit-realization handoff
- IndexedDB persistence
