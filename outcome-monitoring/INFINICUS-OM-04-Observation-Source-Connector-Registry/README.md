# INFINICUS OM-04 — Observation Source and Connector Registry

Version: 1.0.0

OM-04 registers the systems, APIs, files, databases, sensors, and manual evidence sources that provide actual business observations for governed Outcome Monitoring metrics.

Public API:

`window.INFINICUS.OM.observationSourceConnectorRegistryEngine`

## Capabilities

- Observation-source registry
- Connector registry
- Source-to-metric binding
- Source ownership and classification
- Endpoint and credential-reference metadata
- Source freshness and cadence rules
- Data-quality minimums
- Health status tracking
- Environment and region constraints
- Connector capability matching
- OM-05 observation-collection handoff
- IndexedDB persistence
