# INFINICUS OM-06 — Data Quality and Evidence Validation Engine

Version: 1.0.0

OM-06 validates collected observations before they are allowed to influence outcome evaluation.

Public API:

`window.INFINICUS.OM.dataQualityEvidenceValidationEngine`

## Capabilities

- Observation completeness validation
- Freshness validation
- Source consistency validation
- Unit and value-type validation
- Duplicate and replay detection
- Raw-evidence presence validation
- Data-quality scoring
- Reliability scoring
- Confidence adjustment
- Validation issue registry
- Accepted and rejected evidence separation
- OM-07 baseline and target handoff
- IndexedDB persistence
