# INFINICUS OM-14 — Causation Assessment Engine

Version: 1.0.0

OM-14 evaluates whether available evidence supports a causal relationship between the approved action and the observed outcome.

Public API:

`window.INFINICUS.OM.causationAssessmentEngine`

## Capabilities

- Causation-policy registry
- Temporal-order validation
- Mechanism plausibility assessment
- Dose-response evidence
- Counterfactual evidence assessment
- Confounder penalty
- Alternative-explanation penalty
- Reproducibility evidence
- Causal-strength scoring
- Causation classification
- Explicit uncertainty and insufficiency handling
- OM-15 external-factor handoff
- IndexedDB persistence
