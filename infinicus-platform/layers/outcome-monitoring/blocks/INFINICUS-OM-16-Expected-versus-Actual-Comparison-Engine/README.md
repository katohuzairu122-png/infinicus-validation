# INFINICUS OM-16 — Expected-versus-Actual Comparison Engine

Version: 1.0.0

OM-16 compares governed expected outcomes with actual monitored results after accounting for quality, attribution, causation, and external-factor context.

Public API:

`window.INFINICUS.OM.expectedActualComparisonEngine`

## Capabilities

- Comparison-policy registry
- Expected-versus-actual value comparison
- Absolute and percentage gap calculation
- Direction-aware performance comparison
- Acceptable-range evaluation
- Confounder-adjusted interpretation
- Causation-aware interpretation
- Outcome status classification
- Comparison confidence propagation
- OM-17 confidence and reliability handoff
- IndexedDB persistence
