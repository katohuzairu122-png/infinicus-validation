# INFINICUS OM-03 — Metric and KPI Registry

Version: 1.0.0

OM-03 converts validated monitoring-contract metric definitions into governed Outcome Monitoring metrics and KPI records.

Public API:

`window.INFINICUS.OM.metricKPIRegistryEngine`

## Capabilities

- Metric-definition registry
- KPI-definition registry
- Formula and aggregation metadata
- Unit and value-type controls
- Direction and target semantics
- Metric-source bindings
- Metric versioning
- Duplicate-code detection
- Contract-to-metric linkage
- Metric activation and retirement
- OM-04 observation-source handoff
- IndexedDB persistence
