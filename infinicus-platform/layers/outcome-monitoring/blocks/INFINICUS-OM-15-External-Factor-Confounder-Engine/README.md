# INFINICUS OM-15 — External Factor and Confounder Engine

Version: 1.0.0

OM-15 registers, evaluates, and scores external factors and confounders that may have influenced monitored business outcomes.

Public API:

`window.INFINICUS.OM.externalFactorConfounderEngine`

## Capabilities

- External-factor registry
- Confounder registry
- Market, regulatory, operational, seasonal, competitive, and environmental factor support
- Exposure-window validation
- Metric-impact scoring
- Directional impact classification
- Materiality scoring
- Overlap with action and observation windows
- Residual confounding score
- Adjustment recommendations
- OM-16 expected-versus-actual handoff
- IndexedDB persistence
