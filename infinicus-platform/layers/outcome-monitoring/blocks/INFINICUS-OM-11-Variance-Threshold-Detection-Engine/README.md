# INFINICUS OM-11 — Variance and Threshold Detection Engine

Version: 1.0.0

OM-11 detects material deviations between current outcomes, baselines, targets, acceptable ranges, tolerances, and governed thresholds.

Public API:

`window.INFINICUS.OM.varianceThresholdDetectionEngine`

## Capabilities

- Variance-policy registry
- Absolute variance calculation
- Percentage variance calculation
- Baseline variance
- Target variance
- Acceptable-range breaches
- Tolerance breaches
- Progress-threshold detection
- Severity classification
- Breach evidence registry
- Suppression and deduplication
- OM-12 alert and escalation handoff
- IndexedDB persistence
