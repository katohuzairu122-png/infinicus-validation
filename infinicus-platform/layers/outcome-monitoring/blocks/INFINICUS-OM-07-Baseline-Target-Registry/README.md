# INFINICUS OM-07 — Baseline and Target Registry

Version: 1.0.0

OM-07 governs the baselines, targets, acceptable ranges, tolerances, and target semantics used to evaluate observed business outcomes.

Public API:

`window.INFINICUS.OM.baselineTargetRegistryEngine`

## Capabilities

- Baseline registry
- Target registry
- Acceptable range registry
- Tolerance registry
- Direction semantics
- Effective-date controls
- Baseline provenance
- Target provenance
- Versioning
- Conflict detection
- Contract-to-baseline linkage
- Contract-to-target linkage
- OM-08 schedule handoff
- IndexedDB persistence
