# INFINICUS OM-13 — Attribution Evidence Engine

Version: 1.0.0

OM-13 assembles and scores evidence indicating whether an observed business outcome is plausibly attributable to the approved action.

Public API:

`window.INFINICUS.OM.attributionEvidenceEngine`

## Capabilities

- Attribution-policy registry
- Action-to-outcome linkage
- Timing alignment checks
- Scope alignment checks
- Exposure and treatment evidence
- Counterfactual reference registration
- Alternative explanation evidence
- Attribution-strength scoring
- Evidence sufficiency classification
- Correlation-only classification
- OM-14 causation-assessment handoff
- IndexedDB persistence
