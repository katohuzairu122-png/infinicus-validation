# INFINICUS OM-05 — Observation Collection Engine

Version: 1.0.0

OM-05 collects actual business observations from governed OM-04 sources and connectors.

Public API:

`window.INFINICUS.OM.observationCollectionEngine`

## Capabilities

- Collector registry
- Collection-policy registry
- Scheduled and manual collection
- Metric-source binding execution
- Batch and single-value collection
- Timestamp and freshness enforcement
- Source and connector health enforcement
- Observation classification
- Raw evidence preservation
- Idempotent observation intake
- Collection run tracking
- Failure and dead-letter evidence
- OM-06 data-quality handoff
- IndexedDB persistence
