# BI-21 to BI-22 Contract

BI-22 receives:

- reporting handoff ID,
- root-cause analysis ID,
- investigation case,
- signal,
- ranked drivers,
- cause graph,
- unresolved hypotheses,
- supporting context,
- correlation ID.

# BI-22 to BI-23 Contract

BI-22 provides:

- distribution handoff ID,
- report snapshot ID,
- report title,
- executive summary,
- severity,
- audience,
- export formats,
- correlation ID.
