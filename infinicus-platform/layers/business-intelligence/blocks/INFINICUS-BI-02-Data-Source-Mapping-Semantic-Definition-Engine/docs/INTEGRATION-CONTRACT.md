# Integration Contract with Earlier Layers

## Data Acquisition Layer provides

- raw source records,
- source timestamps,
- source identifiers,
- extraction metadata,
- raw schemas.

## Business Operations Layer provides

- transactional business records,
- process states,
- operational entities,
- business ownership context.

## BI-02 provides

- canonical semantic definitions,
- field mappings,
- lineage,
- conversion rules,
- versioned dataset contracts.

## BI-03 receives

- dataset contract ID,
- source-system ID,
- expected grain,
- mapping rules,
- target semantic entity.
