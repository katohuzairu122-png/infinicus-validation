# BI-09 to BI-10 Contract

BI-10 receives:

- calculation handoff ID,
- metric definitions,
- metric versions,
- dependencies,
- dimensions,
- filters,
- units,
- targets,
- warehouse snapshot ID,
- correlation ID.

# BI-10 to BI-11–BI-18 Contract

BI-10 provides:

- calculation run ID,
- metric result IDs,
- metric values,
- dimensional groups,
- time buckets,
- units,
- target status,
- threshold status,
- source row counts,
- correlation ID.
