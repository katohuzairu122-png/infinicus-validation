# BI-08 to BI-09 Contract

BI-09 receives:

- warehouse dataset ID,
- warehouse snapshot ID,
- dataset type,
- grain,
- primary key fields,
- partition fields,
- row count,
- dataset contract ID.

# BI-09 to BI-10 Contract

BI-09 provides:

- calculation handoff ID,
- metric definitions,
- metric versions,
- dependencies,
- dimensions,
- filters,
- units,
- targets,
- warehouse snapshot ID,
- correlation ID.
