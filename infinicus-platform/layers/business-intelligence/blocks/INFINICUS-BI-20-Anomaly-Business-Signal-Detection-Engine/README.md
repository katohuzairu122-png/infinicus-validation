# INFINICUS BI-20 — Anomaly and Business Signal Detection Engine

Version: 1.0.0

BI-20 detects unusual values, abrupt changes, threshold breaches, cross-domain contradictions, and emerging business signals from BI-19 analytical outputs.

## Capabilities

- Detection-rule registry
- Statistical deviation detection
- Z-score anomaly detection
- Moving-baseline comparison
- Sudden-change detection
- Variance severity escalation
- Benchmark breach detection
- Cross-domain contradiction detection
- Signal deduplication and prioritization
- Confidence and severity scoring
- Investigation queue
- BI-21 root-cause handoff
- IndexedDB persistence
- Demo interface and automated tests

## Dependencies

- BI-01 through BI-19

## Public API

`window.INFINICUS.BI.anomalySignalDetectionEngine`
