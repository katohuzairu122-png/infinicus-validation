# INFINICUS ADI-08

Business Goal Registry, version 1.0.0.

ADI-08 manages measurable, versioned business goals used by decision triggers, problem framing and recommendation evaluation. It preserves the existing INFINICUS goal fields while adding tenant isolation, direction-aware progress, lifecycle governance and ADI-04 context publication.

The package does not modify the legacy Goal Registry in the consolidated HTML. Import/export helpers support controlled migration.
