# INFINICUS ADI-10

Business Problem Definition Engine, version 1.0.0.

ADI-10 converts authorized goals, triggers and evidence into a structured decision problem. It preserves the existing Problem Framing fields while adding tenant isolation, reference validation, immutable versions, problem boundaries and ADI-04 context publication.

The engine defines the decision problem only. Candidate actions begin in ADI-13.
