# INFINICUS ADI-12

Decision Objectives, Constraints and Criteria Engine, version 1.0.0.

ADI-12 converts a ready ADI-11 analysis case into a governed `DecisionEvaluationFramework`. It records decision-specific objectives, hard and soft constraints, evaluation criteria, units, directions and stakeholder-supplied weights.

ADI-12 defines how alternatives will later be evaluated. It does not generate alternatives or calculate option scores.
