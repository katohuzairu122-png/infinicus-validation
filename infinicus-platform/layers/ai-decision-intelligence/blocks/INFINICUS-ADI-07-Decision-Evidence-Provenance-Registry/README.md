# INFINICUS ADI-07

Decision Evidence and Provenance Registry, version 1.0.0.

ADI-07 converts authorized context fragments into immutable, hash-verifiable evidence records. It preserves source identity, acquisition time, quality, freshness and parent evidence links while maintaining tenant/business/decision isolation.

Evidence content is append-only. Supersession and revocation are recorded as separate lifecycle entries.
