# INFINICUS ADI-09

Decision Trigger Registry, version 1.0.0.

ADI-09 records manual and system-generated signals that may require a business decision. It preserves the existing INFINICUS trigger fields while adding tenant isolation, goal/evidence links, deduplication, lifecycle rules, immutable versions and ADI-04 context publication.

Recording a trigger does not automatically approve an action or define a business problem.
