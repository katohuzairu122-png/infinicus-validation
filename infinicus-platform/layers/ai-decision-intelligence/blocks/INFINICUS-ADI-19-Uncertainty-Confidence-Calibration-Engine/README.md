# ADI-19 — Uncertainty, Confidence and Calibration Engine

Produces per-alternative confidence estimates from scoring coverage, simulation completeness, risk quantification and sensitivity stability. Calibration is explicit and versioned; rankings are never modified.

## Runtime service

`adi.confidence_calibration`

Routes: `adi.confidence.assess`, `adi.confidence.get`.

See `docs/INTEGRATION-CONTRACT.md` for attachment requirements.
