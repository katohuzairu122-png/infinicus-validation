# INFINICUS ADI-06

Simulation Engine Results Adapter, version 1.0.0.

ADI-06 is a read-only boundary between the existing INFINICUS Simulation Engine and AI Decision Intelligence. It retrieves completed simulation runs, validates ownership and run integrity, and maps their recorded outputs into provenance-preserving ADI-04 context fragments.

ADI-06 cannot start, rerun, alter or reinterpret a simulation.
