# ADI-24 — Approved Business Action Package Publication and Handoff

Builds an integrity-protected Decision Package from an ADI recommendation endorsed for ABA handoff. A supplied publisher transports the package to ABA; it does not approve or execute the underlying business action.

Service: `adi.aba_handoff`; routes: `adi.handoff.build`, `adi.handoff.publish`, `adi.handoff.get`.
