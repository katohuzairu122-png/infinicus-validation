# INFINICUS ADI-02

Decision Request Intake and Validation Engine, version 1.0.0.

This block converts an authorized problem, opportunity, warning, alert or reassessment request into a canonical `DecisionCase`. It performs normalization, validation, classification, prioritization, duplicate detection and audit event publication.

ADI-02 never recommends, approves or executes an action.

## Run

```bash
npm test
```
