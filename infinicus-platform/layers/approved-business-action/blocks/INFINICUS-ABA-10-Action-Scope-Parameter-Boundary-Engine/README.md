# INFINICUS ABA-10 — Action Scope, Parameter and Boundary Engine

Version: 1.0.0

ABA-10 converts an issued approved-action contract into enforceable scope, parameter, timing, quantity, value, geography, target, and execution boundaries.

Public API:

`window.INFINICUS.ABA.actionScopeBoundaryEngine`

## Capabilities

- Boundary-policy registry
- Target-scope controls
- Parameter minimum and maximum controls
- Value and quantity limits
- Geographic limits
- Time-window limits
- Allowed and forbidden operation lists
- Approval-condition enforcement
- Out-of-bound request rejection
- Boundary verification
- ABA-11 revalidation handoff
- IndexedDB persistence
