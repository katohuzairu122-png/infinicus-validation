# INFINICUS ABA-03 — Action Definition and Business Action Ontology Engine

Version: 1.0.0

ABA-03 translates accepted AI Decision Intelligence packages into normalized business-action definitions.

Public API:

`window.INFINICUS.ABA.actionDefinitionOntologyEngine`

## Capabilities

- Action type registry
- Action category registry
- Parameter schema registry
- Target type registry
- Reversibility classification
- Execution adapter requirements
- Approval and monitoring requirement metadata
- Recommendation-to-action mapping
- Unsupported action quarantine
- ABA-04 action-instance handoff
- IndexedDB persistence
