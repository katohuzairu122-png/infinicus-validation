# INFINICUS DT-02 — Business Identity and Twin Instance Registry

Version: 1.0.0

DT-02 creates and governs Digital Twin instances for businesses, branches, projects, departments, and other approved business units.

## Capabilities

- Business identity registry
- Twin instance creation
- Parent-child twin hierarchy
- Branch, project, department, and unit twins
- Unique twin keys and aliases
- Ownership and tenancy metadata
- Lifecycle status management
- Twin activation, suspension, and retirement
- Business-to-twin mapping
- Twin discovery and lookup
- IndexedDB persistence
- DT-03 schema handoff
- Demo interface and automated tests

## Dependencies

- DT-01 — Digital Twin Core Runtime and Registry

## Public API

`window.INFINICUS.DT.twinInstanceRegistry`
