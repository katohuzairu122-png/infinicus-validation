# INFINICUS DT-08 — Customer and Demand Twin Engine

Version: 1.0.0

DT-08 represents customers, segments, demand, retention, churn, satisfaction, engagement, lifetime value, and customer behavior inside the Business Digital Twin.

## Capabilities

- Customer profile registry
- Customer segment registry
- Demand-state registry
- Retention and churn state
- Engagement and purchase-frequency state
- Customer lifetime value state
- Satisfaction and advocacy state
- Cohort membership
- Demand concentration analysis
- Customer-state confidence and provenance
- Customer and demand snapshots
- DT-09 sales and revenue-flow handoff
- IndexedDB persistence
- Demo interface and automated tests

## Dependencies

- DT-01 through DT-07

## Public API

`window.INFINICUS.DT.customerDemandTwinEngine`
