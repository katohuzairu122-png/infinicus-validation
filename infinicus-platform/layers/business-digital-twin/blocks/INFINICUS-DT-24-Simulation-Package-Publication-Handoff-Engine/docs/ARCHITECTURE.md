# Architecture

1. Receive DT-23 simulation handoff.
2. Load simulation-package policy.
3. Validate readiness and contract completeness.
4. Verify historical and scenario checksums.
5. Validate initial-condition types.
6. Partition actual, assumed, and simulated state.
7. Version the package.
8. Generate reproducibility manifest.
9. Publish package or record rejection.
10. Create final Simulation Engine handoff.
